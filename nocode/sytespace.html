About the project:
SyteSpace provides everyone with a safe, versatile platform enabling them to create their own Minecraft server for free without breaking the bank. My work with the project consisted of creating the bot through witch customers can order their servers. The bot turned out to do more than its main function serving also as a moderation (once I added overwatch) and community bot. It also currently auto-creates account by using the pterodactyl panel API
